# 📝 Console Notepad

Простой консольный блокнот на Java, написанный в учебных целях.

## Описание

Программа позволяет сохранять текстовые заметки и выводить их прямо в консоли. Никаких сторонних зависимостей — только стандартная библиотека Java.

## Команды

| Команда       | Описание                          |
|---------------|-----------------------------------|
| `help`        | Список доступных команд           |
| `saved`       | Сохранить новую заметку           |
| `take out`    | Вывести заметку по ID             |
| `take out all`| Вывести все сохранённые заметки   |
| `exit`        | Выйти из программы                |
| `clear`       | очистить лист                     |

## Запуск

Никаких сборщиков и зависимостей не нужно.

1. Откройте проект в любом редакторе (IntelliJ IDEA, VS Code, Eclipse и т.д.)
2. Запустите файл `Main.java`

> Убедитесь, что файл `comands.java` находится в папке `clases/` рядом с `Main.java`.

## Структура проекта

```
├── Main.java         # Точка входа, основной цикл программы
└── clases/
    └── comands.java  # Вспомогательные методы (help, вывод ошибок)
```

## Заметки

Это учебный проект — данные хранятся только в памяти и не сохраняются между запусками.
